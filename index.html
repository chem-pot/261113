⚗️ 알칼리 금속 + 페놀프탈레인 용액 반응 시뮬레이션

페놀프탈레인 용액이 들어있는 물에 리튬·나트륨·칼륨을 동시에 넣으면?

🧪 페놀프탈레인 지시약 : 산성·중성 → 무색  |  염기성(pH 8.2↑) → 선명한 분홍색
반응이 격렬할수록 OH⁻ 이온이 빠르게 생성되어 색이 더 빨리 진해져!
2Li + 2H₂O → 2LiOH + H₂↑
천천히 분홍색으로 변해
2Na + 2H₂O → 2NaOH + H₂↑
빠르게 진한 분홍색으로 변해
2K + 2H₂O → 2KOH + H₂↑
즉시 매우 진한 분홍색!

pH : 7.0
